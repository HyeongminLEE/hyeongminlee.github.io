Ensemble-based Convolutional Neural Network for Facial Emotion Recognition

Jinkyung Do, Hanbyeol Bae, Jaesung Choi, Hyeongmin Lee, Sangyoun Lee

Abstract

Recently, recognizing the facial emotion task has shown to be a challenging topic. Because it is difficult to generalize each expressions in the wild condition. In this paper, we proposed improved facial emotion recognition method by adopting the improved inception module and the ensemble approach. To achieve this, we trained models based on inception module and models using well-known effective architectures. Then, we selected top 10 models and averaged for voting result.
